QR Code-Based Smart Student ID Generator

A Python application that automatically creates colorful student ID cards from a CSV file. Each card embeds a QR code, college logo, optional student photo, and can be exported to PNG and PDF formats.

Features:

  • Reads student data (name, registration number, department, year, phone) from student_data.csv.
  • Generates QR codes with student details using the qrcode library.
  • Adds college logo and themed header using Pillow (PIL).
  • Optionally inserts a student photo for specific registration numbers.
  • Exports every card as PNG and PDF using reportlab.
  • Easy customization of colors, fonts, and layout.

Tech Stack:

Tool / Library Purpose
Python 3.10 Core programming language
pandas Load & iterate CSV rows
Pillow (PIL) Build card image, draw text, paste images
qrcode QR code generation
reportlab PDF export
